About Crafty Flips

Crafty Flips is a personal Python-based command-line tool for managing and scheduling content uploads across multiple social media platforms including YouTube, Facebook, Instagram, and TikTok.

This is a developer-centric tool designed for individuals who understand API integration and are comfortable configuring platform credentials. It runs locally on your machine and provides a streamlined workflow for cross-platform content management and analytics tracking.

Current Features: Multi-platform upload scheduling, analytics sync, performance tracking, and cross-platform video management for personal use.

Privacy Policy

Last updated: December 2025

Crafty Flips is a personal developer tool that runs locally on your machine. This privacy policy explains how the tool handles data when you use it.

Information Collection and Storage

Crafty Flips does not collect, store, or transmit any personal information to external servers. All data is stored locally on your machine in a SQLite database file.

OAuth & Platform API Access

When you configure Crafty Flips to connect with social media platforms (YouTube, Facebook, Instagram, TikTok), you provide OAuth access tokens or API credentials. These credentials:

  • Are stored locally in your config.json file on your machine
  • Are used solely to perform actions you explicitly request (uploading videos, fetching analytics)
  • Are never transmitted to any third-party servers except the respective platform APIs
  • Are your responsibility to keep secure (standard file permissions apply)

Third-Party Platform Integrations

When you use Crafty Flips to upload content or fetch analytics from platforms like TikTok, YouTube, Facebook, or Instagram, those platforms may collect data according to their own privacy policies. Crafty Flips only acts as an intermediary tool executing your commands.

Local Data Storage

The tool stores the following data locally on your machine:

  • Video metadata (titles, descriptions, file paths)
  • Upload records and scheduling information
  • Analytics data fetched from platform APIs
  • Platform credentials (in config.json)

Data Sharing

Crafty Flips does not share, sell, or distribute any data. The tool operates entirely on your local machine and only communicates directly with platform APIs you configure.

Security Recommendations

Since this is a local tool, security is your responsibility:

  • Protect your config.json file with appropriate file permissions
  • Do not commit credentials to version control systems
  • Regularly rotate API tokens and access credentials
  • Keep your local machine secure with standard security practices
